# Development Environment

This guide provides detailed instructions for setting up a complete local development environment for Clamo.

## Prerequisites

### Required Software

| Software | Version | Purpose |
|----------|---------|---------|
| Node.js | 20+ | Frontend and TypeScript services |
| pnpm | 8+ | Package manager |
| Python | 3.11+ | CEJ connector and AI agents |
| uv | Latest | Python package manager |
| Go | 1.22+ | Ingestion worker |
| Rust | Latest | clamo-sync and dashboards |
| Docker | Latest | Containerized services |
| Git | Latest | Version control |

### Installing Prerequisites

<CodeGroup>

```bash macOS
# Homebrew
brew install node pnpm python uv go rust docker git

# Verify installations
node --version && pnpm --version && python --version
uv --version && go version && rustc --version
```

```bash Ubuntu/Debian

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
nvm install 20

npm install -g pnpm

sudo apt install docker.io docker-compose
```

## Clone Repositories

### Complete Clone Script

```bash
#!/bin/bash
# clone-all.sh

GITHUB_ORG="GetClamo"
BASE_DIR="$HOME/clamo"

mkdir -p $BASE_DIR
cd $BASE_DIR

repos=(
  "clamo-web-app"
  "clamo-tenant"
  "clamo-cases"
  "clamo-database"
  "clamo-commons"
  "clamo-cej-connector"
  "clamo-ingest-go"
  "clamo-sync"
  "clamo-mailing"
  "clamo-dashboards"
  "ask-clamo"
  "clamo-setup"
  "docs"
)

for repo in "${repos[@]}"; do
  if [ ! -d "$repo" ]; then
    echo "Cloning $repo..."
    git clone "git@github.com:$GITHUB_ORG/$repo.git"
  else
    echo "$repo already exists, pulling latest..."
    cd $repo && git pull && cd ..
  fi
done

echo "Done! All repositories cloned to $BASE_DIR"
```

## Environment Variables

### Main .env File

Create a `.env` file in the Clamo root directory:

```bash
# ===========================================
# DATABASE
# ===========================================

# Control Database (Supabase)
CONTROL_DATABASE_URL=postgresql://postgres:[PASSWORD]@db.[PROJECT].supabase.co:5432/postgres

# Neon (for tenant provisioning)
NEON_API_KEY=neon_api_key_here
NEON_PROJECT_ID=project_id_here

# ===========================================
# AUTHENTICATION (WorkOS)
# ===========================================

WORKOS_API_KEY=sk_live_...
WORKOS_CLIENT_ID=client_...
WORKOS_REDIRECT_URI=http://localhost:3001/callback

# ===========================================
# SUPABASE (Vault and Storage)
# ===========================================

SUPABASE_URL=https://[PROJECT].supabase.co
SUPABASE_SERVICE_KEY=eyJ...

# ===========================================
# EXTERNAL SERVICES
# ===========================================

# Anthropic (LLM)
ANTHROPIC_API_KEY=sk-ant-...

# SendGrid (Email)
SENDGRID_API_KEY=SG....

# Ably (Real-time)
ABLY_API_KEY=...

# ===========================================
# KAFKA
# ===========================================

KAFKA_BROKERS=localhost:9092
```

## Clamo Setup CLI

### Installation

```bash
cd ~/clamo/clamo-setup
uv tool install . --force
```

### Initial Configuration

```bash
cd ~/clamo
clamo dev setup
```

### Useful Commands

```bash
# Start all services
clamo dev start

# Start with Docker rebuild
clamo dev start --rebuild

# Stop services
clamo dev stop

# View status
clamo dev status

# Switch to local commons
clamo commons use-local

# Switch back to remote commons
clamo commons use-remote
```
