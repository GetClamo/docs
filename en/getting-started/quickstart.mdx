# Quickstart

This guide will help you make your first integration with Clamo APIs.

## Prerequisites

Before starting, you need:

- Clamo access credentials (provided by your administrator)
- A valid WorkOS session token
- Node.js 20+ or Python 3.10+ (depending on your stack)

## Step 1: Obtain Authentication Token

Clamo uses WorkOS AuthKit for authentication. The session token is obtained after login:

<CodeGroup>

```javascript JavaScript
import { getSession } from '@workos-inc/authkit-nextjs';

const session = await getSession();
const token = session?.accessToken;
```

```python Python
# The token is obtained from the WorkOS authentication flow
# and passed in the x-clamo-session header
import base64
import json

session_data = {
    "userId": "user_123",
    "organizationId": "org_456",
    "email": "user@company.com"
}
token = base64.urlsafe_b64encode(json.dumps(session_data).encode()).decode()
```

</CodeGroup>

## Step 2: Configure Headers

All requests to Clamo require the following headers:

```bash
Authorization: Bearer <workos_access_token>
x-clamo-session: <base64_encoded_session>
Content-Type: application/json
```

## Step 3: Get User Information

<CodeGroup>

```javascript JavaScript
const response = await fetch('https://api.clamo.dev/v1/me', {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'x-clamo-session': sessionToken,
  },
});

const user = await response.json();
console.log(user);
// {
//   id: "usr_abc123",
//   email: "user@company.com",
//   firstName: "John",
//   lastName: "Doe",
//   role: "admin",
//   companyId: "cmp_xyz789"
// }
```

```python Python
import httpx

response = httpx.get(
    "https://api.clamo.dev/v1/me",
    headers={
        "Authorization": f"Bearer {access_token}",
        "x-clamo-session": session_token,
    }
)

user = response.json()
print(user)
```

```bash cURL
curl -X GET https://api.clamo.dev/v1/me \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  -H "x-clamo-session: $SESSION_TOKEN"
```

</CodeGroup>

## Step 4: List Cases

Once authenticated, you can get your company's cases:

<CodeGroup>

```javascript JavaScript
const response = await fetch('https://api.clamo.dev/v1/cases', {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'x-clamo-session': sessionToken,
  },
});

const { data: cases, pagination } = await response.json();

cases.forEach(caso => {
  console.log(`${caso.expediente} - ${caso.demandante} vs ${caso.demandado}`);
  console.log(`  Status: ${caso.status}`);
  console.log(`  Risk: ${caso.riskLevel}`);
});
```

```python Python
response = httpx.get(
    "https://api.clamo.dev/v1/cases",
    headers={
        "Authorization": f"Bearer {access_token}",
        "x-clamo-session": session_token,
    }
)

data = response.json()
for caso in data["data"]:
    print(f"{caso['expediente']} - {caso['demandante']} vs {caso['demandado']}")
```

```bash cURL
curl -X GET "https://api.clamo.dev/v1/cases" \
  -H "Authorization: Bearer $ACCESS_TOKEN" \
  -H "x-clamo-session: $SESSION_TOKEN"
```

</CodeGroup>

## Step 5: Get Case Details

<CodeGroup>

```javascript JavaScript
const caseId = 'case_abc123';

const response = await fetch(`https://api.clamo.dev/v1/cases/${caseId}`, {
  headers: {
    'Authorization': `Bearer ${accessToken}`,
```

```python Python
case_id = "case_abc123"

response = httpx.get(
```

## Complete Example

Here's a complete integration example:

```javascript
// clamo-client.js
class ClamoClient {
  constructor(accessToken, sessionToken) {
    this.baseUrl = 'https://api.clamo.dev';
    this.headers = {
      'Authorization': `Bearer ${accessToken}`,
      'x-clamo-session': sessionToken,
      'Content-Type': 'application/json',
    };
  }

  async getMe() {
    const response = await fetch(`${this.baseUrl}/v1/me`, {
      headers: this.headers,
    });
    return response.json();
  }

  async getCompany() {
    const response = await fetch(`${this.baseUrl}/v1/me/company`, {
      headers: this.headers,
    });
    return response.json();
  }

  async listCases(params = {}) {
    const query = new URLSearchParams(params).toString();
    const response = await fetch(`${this.baseUrl}/v1/cases?${query}`, {
      headers: this.headers,
    });
    return response.json();
  }

  async getCase(caseId) {
    const response = await fetch(`${this.baseUrl}/v1/cases/${caseId}`, {
      headers: this.headers,
    });
    return response.json();
  }

  async getCaseMovements(caseId) {
    const response = await fetch(`${this.baseUrl}/v1/cases/${caseId}/movements`, {
      headers: this.headers,
    });
    return response.json();
  }
}

// Usage
const client = new ClamoClient(accessToken, sessionToken);
const user = await client.getMe();
const cases = await client.listCases({ status: 'EN_TRAMITE' });
```

## Main Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/v1/me` | Current user information |
| GET | `/v1/me/company` | Company information |
| GET | `/v1/cases` | List cases |
| GET | `/v1/cases/:id` | Case details |
| GET | `/v1/cases/:id/movements` | Case movements |
| GET | `/v1/cases/:id/milestones` | Procedural milestones |
| GET | `/v1/cases/:id/progress` | Progress by stages |
