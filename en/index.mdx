title: "Clamo Documentation"
description: "Internal documentation for Clamo, the legal case management platform for the Peruvian labor market"
---

Multi-tenant SaaS platform for labor legal case management in Peru. Automates judicial case tracking from CEJ, integrates AI assistants, and provides real-time analytics.

<Note>
This documentation is for internal use by the Clamo engineering team. For API access and integration questions, contact the platform team.
</Note>

## Platform Capabilities

<CardGroup cols={3}>
  <Card title="Automated Case Tracking" icon="radar">
    Continuous monitoring of CEJ cases with automatic extraction of movements, decisions, and attachments.
  </Card>
  <Card title="Multi-Tenant Architecture" icon="building">
    Database-per-tenant isolation using Neon PostgreSQL branches for security and compliance.
  </Card>
  <Card title="AI Assistants" icon="robot">
    Three specialized assistants (Javi, Clamy, Luna) built with LangGraph for case queries and analytics.
  </Card>
  <Card title="Real-Time Dashboards" icon="chart-line">
    Analytics and metrics powered by Tinybird and ClickHouse for instant insights.
  </Card>
  <Card title="Trusted Sources" icon="shield-check">
    Multi-origin data model with automatic prioritization: manual > AI > CEJ.
  </Card>
  <Card title="Event-Driven Pipeline" icon="bolt">
    Kafka and Temporal-based workflows for reliable data processing.
  </Card>
</CardGroup>
