# Legal Management Platform

Clamo is a multi-tenant SaaS platform designed for comprehensive management of labor legal cases in Peru. It automates judicial case tracking, integrates specialized AI assistants, and provides real-time analytics.

## Key Features

<CardGroup cols={2}>
  <Card title="Automated Tracking" icon="radar">
    Automatic monitoring of CEJ (Electronic Justice Center) cases with real-time updates.
  </Card>
  <Card title="Multi-Tenancy" icon="building">
    Database-per-tenant architecture with Neon PostgreSQL for complete data isolation.
  </Card>
  <Card title="AI Assistants" icon="robot">
    Specialized agents with LangGraph for case queries and predictive analysis.
  </Card>
  <Card title="Advanced Analytics" icon="chart-line">
    Real-time dashboards and metrics with Tinybird and ClickHouse.
  </Card>
</CardGroup>

## Technology Stack

| Layer | Technologies |
|-------|--------------|
| **Frontend** | Next.js 16, React 19, TypeScript |
| **APIs** | Hono, Rust/Axum |
| **Database** | Neon PostgreSQL, Prisma ORM |
| **Analytics** | Tinybird, ClickHouse |
| **AI** | LangGraph, LangChain, Anthropic |
| **Authentication** | WorkOS AuthKit |
| **Messaging** | Kafka, Temporal |
