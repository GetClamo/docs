# Plataforma de Gestión Legal

Clamo es una plataforma SaaS multi-tenant diseñada para la gestión integral de casos legales laborales en Perú. Automatiza el seguimiento de expedientes judiciales, integra asistentes de IA especializados y proporciona análisis en tiempo real.

## Características Principales

<CardGroup cols={2}>
  <Card title="Seguimiento Automatizado" icon="radar">
    Monitoreo automático de expedientes del CEJ (Centro de Justicia Electrónico) con actualizaciones en tiempo real.
  </Card>
  <Card title="Multi-Tenancy" icon="building">
    Arquitectura de base de datos por tenant con Neon PostgreSQL para aislamiento completo de datos.
  </Card>
  <Card title="Asistentes IA" icon="robot">
    Agentes especializados con LangGraph para consultas de casos y análisis predictivo.
  </Card>
  <Card title="Analytics Avanzado" icon="chart-line">
    Dashboards y métricas en tiempo real con Tinybird y ClickHouse.
  </Card>
</CardGroup>

## Pila Tecnológica

| Capa | Tecnologías |
|------|-------------|
| **Frontend** | Next.js 16, React 19, TypeScript |
| **APIs** | Hono, Rust/Axum |
| **Base de Datos** | Neon PostgreSQL, Prisma ORM |
| **Analytics** | Tinybird, ClickHouse |
| **IA** | LangGraph, LangChain, Anthropic |
| **Autenticación** | WorkOS AuthKit |
| **Mensajería** | Kafka, Temporal |
