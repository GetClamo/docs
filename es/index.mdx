---
title: "Documentación Clamo"
description: "Documentación interna de Clamo, la plataforma de gestión de casos legales laborales para el mercado peruano"
---

Plataforma SaaS multi-tenant para gestión de casos legales laborales en Perú. Automatiza el seguimiento de expedientes del CEJ, integra asistentes IA y proporciona analíticas en tiempo real.

<Note>
Esta documentacion es para uso interno del equipo de ingenieria de Clamo. Para preguntas sobre acceso a la API e integracion, contacta al equipo de plataforma.
</Note>

## Capacidades de la Plataforma

<CardGroup cols={3}>
  <Card title="Seguimiento Automatizado" icon="radar">
    Monitoreo continuo de casos del CEJ con extraccion automatica de movimientos, resoluciones y anexos.
  </Card>
  <Card title="Arquitectura Multi-Tenant" icon="building">
    Aislamiento de base de datos por tenant usando branches de Neon PostgreSQL para seguridad y cumplimiento.
  </Card>
  <Card title="Asistentes IA" icon="robot">
    Tres asistentes especializados (Javi, Clamy, Luna) construidos con LangGraph para consultas de casos y analiticas.
  </Card>
  <Card title="Dashboards en Tiempo Real" icon="chart-line">
    Analiticas y metricas impulsadas por Tinybird y ClickHouse para insights instantaneos.
  </Card>
  <Card title="Trusted Sources" icon="shield-check">
    Modelo de datos multi-origen con priorizacion automatica: manual > IA > CEJ.
  </Card>
  <Card title="Pipeline Event-Driven" icon="bolt">
    Flujos de trabajo basados en Kafka y Temporal para procesamiento confiable de datos.
  </Card>
</CardGroup>
